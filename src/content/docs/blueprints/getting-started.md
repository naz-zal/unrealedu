# Getting Started with Blueprints

Blueprints are Unreal Engine's visual scripting system. They are useful for gameplay logic, prototyping, editor tooling, UI flows, and designer-facing customization.

## Good Uses

- wiring events
- tuning gameplay values
- connecting assets
- building quick prototypes

Keep complex systems readable by naming variables clearly and splitting repeated logic into functions or components.
