The Unreal Editor is the main workspace for building levels, importing assets, editing Blueprints, configuring projects, and testing gameplay.

## Core Areas

- **Viewport**: Inspect and edit the current level.
- **Content Browser**: Manage assets, folders, and source content.
- **Outliner**: View actors currently placed in the level.
- **Details Panel**: Edit properties for the selected actor or asset.

## Working Habit

Select an object, inspect its details, then trace where it is referenced. This habit makes large Unreal projects easier to understand.
