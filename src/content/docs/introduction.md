# Introduction

Epic Edu is a lightweight Unreal Engine documentation handbook. The goal is to keep explanations direct, searchable, and organized like a book rather than a marketing site.

This boilerplate starts with a small content map. Each page is intentionally short for now so the structure is easy to extend.

## Scope

The site will grow around practical Unreal Engine topics:

- editor fundamentals
- project and asset organization
- Blueprint workflows
- C++ gameplay programming
- rendering, materials, animation, and systems work

## Style

Pages should prefer concrete examples, short explanations, and code or Blueprint-oriented notes where they help.
